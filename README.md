Maleci GPU Task Manager
=========

How to launch
--------------------

- open project as wsl project in vscode 
- in console run:
    ```npm install```
- in console run:
    ```sudo "$(which node)" server.js```
- in the same directory but in another console run:
    ```sudo "$(which node)" executor.js```
